Logical agents

Think rationally

Until now, the focus has been on agents that act rationally.

Often, however, rational action requires rational (logical) thought on the agent‚Äôs part.

To that purpose, portions of the world must be represented in a knowledge base (KB).

  • Composed of sentences in a language with a truth theory (logic), i.e. sentences can be interpreted as statements about the world (semantics)
  • The sentences themselves have a causal influence on the agent‚Äôs behaviour in a way that is correlated with the contents of the sentences (syntax)
  • Interaction with the KB through ASK and TELL operations (simplified)
    • ASK: asks the KB what action it should perform
    • TELL: tells the knowledge base what it perceives

Search limitations

The problem-solving agents based on search algorithms know things, but only in a very limited, inflexible sense (Russel and Norvig 2022, 226):

  • They know what actions are available (ACTIONS(s)) and what the result of performing a specific action from a speicifc state will be (RESULT(s,a). But they don‚Äôt know general facts.
  • Using atomic represtations, representing knowledge about the environment (i.e., the state space) requires to list all possible concrete states
  • The goal can formally only be described as an explicit set of states
